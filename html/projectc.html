Website voor Centre of Experise

In dit project werd voor het eerst gebruik gemaakt van een realistische opdrachtgever, namelijk het center of Expertise. Tijdens de opdracht heb ik het design proces doorlopen, tot een betaversie van een website. Vanuit de opdrachten heb ik een doelgroep toegewezen gekregen, namelijk docenten. Dit is de eerste website die ik heb gecodeerd.

Doelen

Het project bestond uit een aantal doelen, zoals correct gebruik van html, css en javascript. Het doel was om op de juiste manieren commentaar toe te voegen, zodat iedereen de code zou kunnen snappen. Het volgende doel was om content van een concept tot een detailontwerp uit te werken. Daarbij moet worden onderbouwd hoe de informatie gestructureerd is, door gebruik te maken van een flowchart.

Onderzoek

Vanuit de opdracht ben ik begonnen met user stories van de doelgroep (docenten) en de opdrachtgever (Center of Expertise) uit te werken. Vanuit de user stories heb ik systeemeisen opgesteld en onderverdeeld in functionele en niet-functionele eisen. Daarna heb ik design principles opgesteld, waaruit ik wireframes ben gaan maken.

conceptualiseren

In axure ben ik de wireframes gaan uitwerken tot een interactieve mockup. Als ondersteuning heb ik een styleguide en een flowchart gemaakt. Na feedback heb ik een aantal elementen aangepast.

Lanceren

In deze fase heb ik de interactieve mockup uitgewerkt in een betaversie website. Alle pagina's waren goed gelukt, behalve bij de inschrijfbuttons op de evenementenpagina. In het onderstaande filmpje is het eindresultaat te zien en hoe de gebruiker door de website navigeert.

Reflectie

Dit was mijn eerste beta website en als ik kijk naar de structuur dan vind ik het goed gelukt. Ik zou voor andere kleuren gaan, omdat het contrast tussen sommigen elementen te klein is. Daardoor kan je teksten niet goed lezen, wat niet gebruikersvriendelijk is. Sommige elementen zijn in de opmaak niet helemaal gelukt, maar gelukkig weet ik nu wel hoe dit moet!
