Responsive betawebsite van een tijdlijn

Tijdens het vak IxD & Technology heb ik geleerd hoe ik vanaf een mobile first aanpak een ontwerp kan creÃ«ren en daaruit een responsive ontwerp kan uitwerken met behulp van flexbox en media queries. Ik heb twee betaversies van een website gemaakt, namelijk een tijdlijn en een herontwerp van een fruitboer webshop. Ik zal mij tijdens deze studiecase richten op de tijdlijn, maar de webshop kun je hier bekijken.

Doelen

Het doel van deze opdracht was om eerst affiniteit te krijgen met flexbox, daarna werd dit uitgebreid naar een grid container. Ook was het doel om een responsive ontwerp te maken, dus heb ik geleerd hoe ik met media queries moest werken.

Onderzoek

Voor de tijdlijn was het belangrijk om eerst de juiste informatie voor de pagina te vinden. Ik wilde een tijdlijn over de digitalisering van non-verbale communicatie maken. Ik heb acht communicatie middelen online verder uitgezocht en uitgewerkt. Ook heb ik bijpassende afbeeldingen gezocht.

conceptualiseren

Eerst heb ik schetsen gemaakt en heb ik verschillende elementen uitgeknipt om te kijken welke hierachie het beste zou zijn. Uiteindelijk heb ik het uiteindelijke concept uitgewerkt in adobe xd.

Lanceren

Vanuit het prototype ben ik gaan coderen in sublime. Ik ben begonnen met een blokkenstructuur, omdat dit voor mij een duidelijk beginpunt geeft in een mobile first ontwerp. Daarna heb ik de afbeeldingen en tekst toegevoegd en deze gestyled. Als laatste kijk heb ik gekeken naar de visuele uitstraling en de code opgeschoond.De website kan je zelf bekijken, maar in de video is de responsiveness van de website te zien.

Reflectie

Ik heb tijdens dit project veel geleerd, waardoor ik nu veel beter kan coderen. Ik pak het maken van een website nu stap voor stap aan, waardoor fouten sneller te vinden zijn. Door flexbox, media queries en grid zien mijn websites er nu beter uit! Voor mijn tweede project, de fruitshop webshop, zou ik de volgende keer meer gebruik maken van de ruimte die ik heb.
