Interface ontwerp voor Festival robotica

Het project draaide om het ontwerpen van interface voor een website voor het festival robotica 2025. Het ontwerp bestaat uit een mobiele en desktop versie. Tijdens het ontwerpen heb ik gebruik gemaakt van een styletile en een moodboard. Het project is voortgeborduurd op visual communication 1, waardoor het logo en het kleurenpallet terugkomt.

Doelen

Het doel van het project was om een overzichtelijk interface ontwerp te maken voor een mobiele en desktop website. De website moet bestaan uit een homepagina en een hot-topics pagina, waar informatie wordt getoond. Het was belangrijk om rekening te houden met de hierachie, uitstraling en cohesie.

Onderzoek

Door visual communication 1 had ik al een bepaalde stijl die ik wilde voortzetten. Met deze kleurensamenstelling in mijn achterhoofd ben ik op pinterest moodboards gaan maken. Vanuit de moodboards heb ik een styletile kunnen maken.

conceptualiseren

Vanuit de moodboards en styletile had ik bepaalde ideeen over de hierachie die ik wilde uitwerken. Ook heb ik gekeken naar wat de belangrijkste punten waren vanuit de opdracht, zoals bijvoorbeeld de mogelijkheid om kaartjes te kopen.

Lanceren

Na de schetsen ben ik verder gaan werken in Figma, waar ik eerst heb gespeeld met verschillende stijlen achtergrond, buttons en icons. Ook heb ik gekeken naar de vorm van de afbeeldingen wat het beste aansloot bij de uitstraling van het festival. Hieronder zie je een video van het eindresultaat, maar je kan het zelf ook uitproberen!

Reflectie

Ik heb tijdens dit vak veel geleerd over het ontwerpen van een website. Als ik terugkijk op dit project zie ik veel verbeterpunten, zoals meer rekening houden met margin en kleurgebruik wat niet het gewenste effect heeft geleverd. Maar ik vind het belangrijk om het verschil te zien van waar ik ben begonnen, vergeleken een nieuw project.
